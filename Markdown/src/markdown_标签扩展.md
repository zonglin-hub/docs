### Markdown 常用 html 标签

```markdown
快捷键 - <kbd>Ctrl + C</kbd>
代码块 - <code>echo "Hello World"</code>
```

快捷键 - <kbd>Ctrl + C</kbd>

### Markdown 折叠内容

HTML `<details>` 标签指定了用户可以根据需要打开和关闭的额外细节。

语法：

```markdown
<details> <summary>Title</summary>

contents ...

</details>
```

内容里面可以嵌套使用 Markdown 语法和 HTML 语法。
