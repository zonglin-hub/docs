# Markdown

Rust 中的异步编程

## 环境要求

异步编程书籍使用 [`mdbook`] 构建，你可以通过 cargo 安装它。

```sh
cargo install mdbook
```

[`mdbook`]: https://github.com/rust-lang/mdBook

## 构建

要生成完整的书籍，请运行 `mdbook build`，它会在 `book/` 目录下生成书籍。

```sh
mdbook build
```

## 开发

在编写过程中，实时查看更改会很方便，`mdbook serve` 会启动一个本地 Web 服务器来提供书籍内容。

```sh
mdbook serve
```
