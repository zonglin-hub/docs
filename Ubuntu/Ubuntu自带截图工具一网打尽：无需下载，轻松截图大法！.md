# Ubuntu自带截图工具一网打尽：无需下载，轻松截图大法！



在Ubuntu操作系统中，用户无需额外下载第三方截图工具，即可轻松实现屏幕截图的需求。以下是几种Ubuntu自带截图方法，让您在无需安装任何额外软件的情况下，轻松捕捉屏幕内容。

## 1. 使用键盘快捷键抓图

Ubuntu的GNOME集成桌面环境提供了便捷的键盘快捷键来抓取屏幕图像。

### 1.1 全屏截图

- 按下 `Alt` + `Print Screen` 组合键，即可抓取整个屏幕的图像。

### 1.2 活动窗口截图

- 按下 `Alt` + `Print Screen` 组合键，然后点击需要截图的活动窗口，即可抓取该窗口的图像。

完成截图后，屏幕上会弹出“保存图片”的对话框，您可以选择保存路径和文件名，或者直接复制到剪贴板。

## 2. 使用GNOME截图工具

GNOME截图工具是Ubuntu自带的一款截图工具，它提供了更多截图选项。

### 2.1 安装GNOME截图工具

在终端中输入以下命令安装：

```bash
sudo apt-get install gnome-screenshot
```

### 2.2 使用GNOME截图工具

安装完成后，您可以通过以下方式打开：

- 在应用程序菜单中选择“截图”。
- 或者按下 `Ctrl` + `Shift` + `A` 快捷键。

GNOME截图工具允许您选择截图类型（全屏、窗口、区域等），并提供了编辑和保存截图的功能。

## 3. 使用Shutter截图软件

Shutter是一款功能强大的截图工具，支持多种截图类型和图像编辑功能。

### 3.1 安装Shutter

在终端中输入以下命令安装：

```bash
sudo apt-get install shutter
```

### 3.2 使用Shutter

安装完成后，您可以通过以下方式打开：

- 在应用程序菜单中选择“Shutter”。
- 或者按下 `Ctrl` + `Alt` + `T` 快捷键。

Shutter提供了丰富的截图选项和图像编辑功能，例如标注、裁剪、旋转等。

## 4. 使用 Flameshot

Flameshot 是一款功能强大且易于使用的屏幕截图软件，它具有图形用户界面和命令行版本。

### 4.1 安装Flameshot

在终端中输入以下命令安装：

```bash
sudo apt install flameshot
```

### 4.2 使用Flameshot

安装完成后，您可以通过以下方式打开：

- 在应用程序菜单中选择“Flameshot”。
- 或者按下 `Ctrl` + `Alt` + `F` 快捷键。

Flameshot 提供了丰富的截图功能，包括截图区域、窗口、全屏等，以及图像编辑功能。

## 总结

通过以上几种方法，Ubuntu用户可以轻松实现屏幕截图。无论是简单的全屏截图，还是复杂的图像编辑，这些内置工具都能满足您的需求。
