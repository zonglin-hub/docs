在 Ubuntu 系统中，两种主要方法查看系统启动日志：
---

### 使用 `journalctl` 命令
`journalctl` 是由 systemd 提供的命令，用于查看系统日志，包括启动日志。以下是相关命令：
1. **查看所有启动日志**：
   ```bash
   sudo journalctl -b
   ```
   - `-b` 参数表示从系统启动时开始的日志。

1. **查看最新的启动日志**：
   ```bash
   sudo journalctl -b -0
   ```
   - `-0` 参数表示显示最近一次启动的日志。
3. **查看详细的启动日志**：
   ```bash
   sudo journalctl -b -0 -xe
   ```
   - `-xe` 参数表示显示详细的执行信息和错误信息。
4. **实时查看启动日志**：
   ```bash
   sudo journalctl -b -0 -f
   ```
   - `-f` 参数表示实时跟踪新日志的输出。
---
### 查看传统日志文件
Ubuntu 系统的传统日志文件存储在 `/var/log` 目录中，您可以通过直接查看这些文件来获取启动日志：
1. **查看启动日志文件**：
   ```bash
   sudo cat /var/log/boot.log
   ```
   - `boot.log` 文件通常包含启动相关的日志信息。
2. **实时查看最新的启动日志**：
   ```bash
   sudo tail -f /var/log/boot.log
   ```
   - `tail -f` 命令可以实时显示最新的日志条目。
3. **其他相关日志文件**：
   - **系统日志**：`/var/log/syslog`（包含系统整体日志信息）
     ```bash
     sudo tail -f /var/log/syslog
     ```
   - **内核日志**：`/var/log/kern.log`（包含内核相关的日志信息）
     ```bash
     sudo tail -f /var/log/kern.log
     ```
