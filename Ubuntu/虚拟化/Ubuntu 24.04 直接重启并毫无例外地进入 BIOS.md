在 Ubuntu 24.04 中，虽然没有一个“万能”命令能保证在所有电脑上**直接重启并毫无例外地进入 BIOS**，但有一个命令可以**告诉系统“下次重启时请尝试进入 BIOS 设置”**。最推荐你使用这个现代且高效的方法：

### ⚡ 首选命令：systemctl reboot --firmware-setup

在终端中输入以下命令：

```bash
sudo systemctl reboot --firmware-setup
```

*   **命令作用**：这个命令会**立即重启你的电脑**，并向系统固件（UEFI/BIOS）发送一个请求，要求它进入设置界面。**能否成功进入，最终取决于你的主板固件是否支持此功能**。绝大多数支持 UEFI 的现代电脑（过去十年内的产品）都可以很好地响应这个命令。
*   **工作原理**：`systemctl` 是管理系统和服务的命令。`reboot` 参数表示重启，`--firmware-setup` 选项则指示系统在重启后尝试进入固件设置。
