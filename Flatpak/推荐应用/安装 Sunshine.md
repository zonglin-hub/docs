## 安装 Sunshine

Moonlight 的 GameStream 主机

提供低延迟的云游戏服务器功能，支持 AMD、Intel 和 Nvidia GPU 进行硬件编码。还提供软件编码。您可以从各种设备上的任何 Moonlight 客户端连接到 Sunshine。提供了一个 Web UI，以允许从您最喜欢的 Web 浏览器进行配置和客户端配对。从本地服务器或任何移动设备配对。

注意： Sunshine 需要额外的安装步骤 （Flatpak）。
在安装之前，请确保遵循 [Linux 发行版的设置指南](https://flathub.org/en/setup)

```bash
flatpak install flathub dev.lizardbyte.app.Sunshine
```

```bash
flatpak run --command=additional-install.sh dev.lizardbyte.app.Sunshine
```
