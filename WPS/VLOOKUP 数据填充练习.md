# VLOOKUP 数据填充练习

好的，这是一个非常经典的学习场景。为了让你能够直观地练习 VLOOKUP，我为你设计了两
个表格数据：**“表1（目标表-工资表）”** 和 **“表2（源数据表-员工档案）”**。

**场景说明：**

我们手头有一张 **工资表（表1）**，里面只有员工ID和基本工资，缺少员工的“姓名”和
“部门”。我们需要通过 **员工ID** 作为桥梁，从 **员工档案表（表2）** 中把这些信息
抓取过来。

### 第二步：WPS 中 VLOOKUP 操作步骤

现在我们的目标是：**在 Sheet1 中，根据 A 列的 ID，把 Sheet2 中对应的“姓名”和“部门”抓过来。**

#### 1. 匹配“姓名”

在 **Sheet1** 的 **B2** 单元格输入以下公式：

```excel
=VLOOKUP(A2, Sheet2!A:D, 2, 0)
```

**公式详解（敲黑板）：**
*   `A2`：**查找值**。你要找谁？我们要找 A2 这个 ID。
*   `Sheet2!A:D`：**数据表范围**。去哪里找？去 Sheet2 的 A 到 D 列找。
    *   *注意：VLOOKUP 只能在范围的第一列（A列）找 ID，所以 ID 必须在范围的第一列。*
*   `2`：**列序数**。找到后，你要返回第几列的数据？“姓名”在范围 A:D 里的第 **2** 列。
*   `0`：**匹配模式**。0 代表精确匹配，1 代表模糊匹配。绝大多数情况选 **0**。

#### 2. 匹配“部门”

在 **Sheet1** 的 **C2** 单元格输入以下公式：

```excel
=VLOOKUP(A2, Sheet2!A:D, 3, 0)
```

*   唯一的区别是第三个参数变成了 **3**，因为“部门”在源数据表的第 3 列。

### 💡 WPS 操作小贴士（常见错误避坑）：

1.  **绝对引用（$符号）：**
    如果你拖动公式时，查找范围变了（比如变成了 `Sheet2!A5:D9`），说明你需要锁定范围。修改公式为：
    `=VLOOKUP(A2, Sheet2!$A:$D, 2, 0)`
    按 F4 键可以快速添加美元符号 `$`。
2.  **#N/A 错误：**
    如果出现 `#N/A`，说明没找到对应的 ID。原因通常是：
    *   ID 格式不一致（一个是文本，一个是数字）。
    *   ID 前后有空格（可以用 TRIM 函数清洗数据）。
    *   查找范围写错了，ID 不在范围的第一列。
